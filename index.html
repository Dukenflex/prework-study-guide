HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewpoint.
  • The title element defines title for a webpage.
  • The style element defines the CSS styles associated with a webpage.
  • The base element defines the base URL for a page.
  • The link element connects an external resource to the HTML document.
  • The script element embeds client side scripts in the HTML document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An external style sheet allows developers to keep all their CSS rules in a seperate file.
  • Inline CSS applies to a single HTML tag.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch: lists your branches.
  • git pull: fetch and merge any commits from the tracking remote branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tells the computer how to perfrom a certain task.
